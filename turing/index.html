Turing Machine Simulator
Description

A fun programming toy and educational tool, this Haskell program simulates the operation of a Turing machine.

Simply load a JSON file containing a machine definition, provide an input tape, and then start running!

While running, the program will provide a visual of the tape contents, the position of the read/write head, and the machine's state.

Step individually to examine each transition the machine makes, or run continuously to see whether the machine terminates and the input was accepted.

Technology
Haskell
JSON
  • The simulator is built in Haskell, utilizing its functional language features to make the Turing machine both efficient and accurate. Haskell's list comprehension and lazy evaluation features are perfect for implementing an infinite tape.
  • The simulator builds machines from a JSON object with keys defining its states, alphabets, and transitions. Input JSON files are interpreted using the shift-reduce parser, which was written from scratch for the project.
