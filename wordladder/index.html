Word Ladder
Description

Word Ladder is an Android app inspired by daily word games like Wordle. Each day the app loads a new challenge consisting of a starting word and target word.

From the starting word, the player must reach the target word by changing one letter at a time, or rearranging all four, to create new words.

The player is scored by how few steps they take, and they can view the scoreboard to see how other players performed.

Technology
Kotlin
Android
Retrofit
AWS
PHP
  • Word Ladder was developed with Kotlin, taking advantage of its functional language features and concise syntax to cut down on boilerplate code.
  • Building this app provided in-depth experience with the Android ecosystem as well as general patterns of mobile development across operating systems.
  • Retrofit and other REST API libraries were utilized to download daily challenges and post scores to the scoreboard.
  • A web server hosted through AWS acted as the endpoint for the app's API calls.
  • On the server, PHP scripts handled GET and POST requests for daily challenges and score submission.
