Picnix
Description

Born from a love for the puzzles known as "Nonograms" or "Picross", Picnix is a spin-off of this grid-based puzzle game with a picnic-themed twist.

The Picnix engine is robust and extensible, handling game logic, rendering and animation, mouse and keyboard input, and user save data through abstract code interfaces that make building new games easy.

The engine also includes a DOM-like UI layout system with hooks for user input (onClick, onHover, etc.) for easily constructing a wide variety of game screens and interfaces.
