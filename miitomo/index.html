Miitomo Archive
Description

Miitomo was a social app by Nintendo where the user answers questions to share with friends. It shut down due to a dwindling user base, and took down all user-generated content with it.

Before the app closed, I took screen recordings of over a thousand questions and reply threads created between my family and I, with the plan to one day save the data.

By writing custom tools that processed the recordings frame by frame to extract, organize, and sanitize the data, I managed to immortalize our Miitomo memories.

Finally, to provide a convenient way to browse the archived content, I designed a web application with a filtered search tool and a polished UI reminiscent of the original app.

Note: the demo linked above is a static page with the database embedded in JavaScript, whereas the original project used asynchronous requests to retrieve content from a server database.

  • Server-side, PHP scripts are responsible for sending database queries to retrieve and respond with filtered data.
  • A MySQL database stores and manages the relationships between questions, answers, and replies.
  • Utilizing OCR technology to extract text from the processed images was a crucial step in compiling clean data.
