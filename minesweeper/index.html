Minesweeper Rewritten
Description

I'm a longtime fan of the classic Minesweeper game, and since Windows no longer comes packaged with it, I challenged myself to recreate the game in one sitting.

After getting the base game implemented, I added animations, particles, and sound effects to give the game some extra flair. Otherwise, it's plain old Minesweeper.

(If you manage to clear a big pocket, watch out for the flying worm!)

Technology
Java
Multithreading
Graphics
  • The game was programmed entirely from scratch in Java using various built-in Java libraries.
  • This project requires diligent use of multithreading to simultaneously perform game logic, render the screen, and handle user input while balancing performance.
  • Minesweeper Rewritten uses Java's 
    Graphics2D
    library alongside 
    AWT
    to render custom graphics in a native window.
